# Collections 

`Acest topic il abordezi dupa ce termini de studiat din package-ul ObjectClass`

1. Ce este un vector (array)?
   Un vector este un grup de variabile la care se facer referire printr-un nume comun, fiecarui variabila ii corespunde un index;
2. Ce este o lista (List)?
   Este o colectie ordonata de obiecte in care pot fi stocate valori duplicat.Lista este o interfata.
3. ArrayList
   Este o matrice rediminsionabila si este folosita pentru a stoca o colectie de elemente de dimensiuni dinamice.Aceasta isi mareste sau miscoreaza dimensiunea cand se adauga sau se scot elemente.
4. LinkedList
   
5. Array vs. ArrayList vs. LinkedList
   ArrayList foloseste o matrice pentru a stoca elemente iar LinkedList foloseste o lista dubla.
   Arraylist: Manipularea este lenta deoarece foloseste intern o matrice, iar LinkedList este mai rapida deoarece foloseste o lista dubla legata.
   ArrayList: Mai bun pentru stocarea si accesarea datelor, LinkedList mai bun pentru manipularea datelor.
6. Ce este o stiva (stack)
   Este o structura de date liniara fiind utilizata pentru a stoca o colectie de obiecte.Se bazeaza pe principiul LIFO Last in first out.
7. Ce este o coada (queue)
   Colectie de elemente ordonate, se poate folosi pentru a stoca elemente inainte de afi procesate.
8. Ce sunt Map, HashMap
9. Ce este un arbore (tree, binary tree)
tree -este o structura de date neliniara in care obiectele sunt organizate in termeni de relatie ierarhica, datele nu sunt organizate liniar.
   Binary Tree- Inseamna ca nodul poate avea axim doi copii, un nod poate avea 1,2 sau 0 "copii"
