# Intrebari frecvente interviu

0. Care sunt principiile OOP?
-Abstrazctizarea
-Incapsularea
-Mostenirea
-Polimorfism   

1. Ce este incapsularea?
Intelesul incapsularii este de a ne asigura ca datele sensibile
sunt ascunse utilzatorului. Pentru a realiza acest lucru trebuie sa declaram atributele de clasa ca privat. Accesul se poate face prin apelarea unor metode publice get si set.   
   Get returneaza valoarea unei variabile.
   Set seteaza valoarea unei variabile

2. Ce este abstractizarea?
    Abstractizarea datelor este procesul de ascundere a anumitor detalii, dar si de a afisa utilizatorului numai informatiile esentiale.

3. Ce este mostenirea?
Prin conceptul de mostenire permitem unor clase(subclase) sa mosteneasca atribute de la o alta clasa (superclasa).
   Pentru a mosteni dintr-o clasa folosim cuvantul cheie extends.

4. Ce este polimorfismul?
Apare atunci cand avem mai multe clase legate intre ele prin mostenire
   aici avem overloading si overriding
   Overloading supraincarcare -este cand avem mai multe metode cu acelasi nume dar parametri diferiti
   Overriding suprascriere - este cand avem metoda din sub clasa care are acelasi nume si parametri cu metoda din superclasa

6. Care sunt specificatorii de acces? Care sunt nivelele de acces pentru fiecare?
Private- membrul este accesibil doar in interiorul clasei in care este definit.
Public - membrul este accesibil in interiorul oricarei clase indiferent de pachetul din care face parte clasa   
Protected - membrul este accesibil in cadrul aceluiasi pachet dar poate fi accesibil si din cadrul altor clase din pachete diferite dar care mostenesc clasa din care face parte membrul protected   
Package - este by default daca nu specificam un specificator din cei de ma sus. Acesta este accesibil din cadrul tuturor claselor din acelasi pachet.   
7. Ce este o clasa?
Un tip de date definite de utilizator
8. Ce este un obiect?
Este o instanta a unei clase
9. Ce este o instanta?
Este o metoda ce necesita crearea unui obiect din clasa inainte de a fi apelata
10. Care este diferenta dintre o variabila statica si una ne-statica?
Static - Variabila poate fi creata fara a apela un obiect din clasa 
Instance - Variabila nu poate fi apelata fara a creia un obiect din clasa    
11. Ce este "this" si ce este "super"?
this- se refera la obiectul dintr-o metoda
super - se foloseste pentru a face diferenta dintre super clase si subclase care au metode cu acelasi nume
